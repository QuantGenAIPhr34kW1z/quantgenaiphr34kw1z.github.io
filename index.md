# About

I design and implement low-level, high-reliability systems, with an emphasis on correctness, security, and long-term maintainability.

My work spans:
- LLM model training, inference pipelines, and applied AI systems
- applied cryptography, post-quantum cryptography (PQC), secure transport layers, and system hardening
- blockchain architectures, decentralized protocols, and cryptoeconomic systems
- cybersecurity engineering, red-team operations, and security audits
- distributed networks and overlay protocols
- Kubernetes-based infrastructure, automation, and large-scale data pipelines
- Big Data processing and high-performance computing workloads
- research-driven tooling, compilers, and system prototypes

Repositories here include both production-grade work and experimental, OSS-first research exploring system design trade-offs and emerging architectures.
